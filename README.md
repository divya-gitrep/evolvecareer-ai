# 🚀 EvolveCareer AI

EvolveCareer AI is an AI-inspired career transition platform designed to help Software Quality Assurance (QA) professionals navigate the shift toward AI-enabled roles.

Built during the OpenAI x Outskill AI Builders Hackathon, the project helps experienced QA professionals understand where they stand today, identify skill gaps, follow personalized learning roadmaps, practice through mission simulations, and track their progress toward AI-enabled careers.

---

## 🎯 Problem Statement

As AI continues to transform the software industry, many QA professionals face questions such as:

* Where do I start learning AI?
* Which AI skills are relevant to my career?
* How can I transition without starting over?
* Will my current skills still matter in the AI era?
* How do I stay motivated throughout the learning journey?

EvolveCareer AI provides a structured and personalized approach to answering these questions.

---

## ✨ Features

### 📝 Assessment

Users can:

* Select their current role:

  * Manual QA Engineer
  * Test Automation Engineer

* Select their target role:

  * AI-Augmented QA Strategist
  * AI-Assisted Test Automation Specialist

* Answer assessment questions:

  * Coding confidence
  * AI familiarity
  * Years of experience

---

### 📊 Personalized Dashboard

Based on assessment inputs, users receive:

* AI Readiness Score
* Transferable Strengths Analysis
* Skill Gap Identification
* Personalized 30-Day Learning Roadmap
* Role-Based Mission Simulations

---

### 🗺️ 30-Day Learning Roadmap

A structured learning plan designed to:

* Reduce information overload
* Focus on relevant skills
* Create achievable milestones
* Improve learning consistency

---

### 🎮 Mission Simulations

Scenario-based exercises that help users:

* Apply newly learned concepts
* Understand target-role responsibilities
* Build confidence through practice

---

### 📈 Progress Tracker

Track:

* Weekly learning progress
* Completed milestones
* Mission simulation progress
* Achievement badges

Includes a lightweight achievement-sharing workflow.

## 🏗️ Technical Architecture

### Frontend

* Next.js (App Router)
* React
* TypeScript
* Tailwind CSS

### Personalization Layer

The MVP uses:

* Local TypeScript data
* Rule-based personalization
* Conditional rendering

User inputs are mapped to predefined career-transition journeys that generate:

* Readiness insights
* Learning roadmaps
* Skill-gap recommendations
* Mission simulations

### Backend

* No dedicated backend services (MVP)

### Database

* No database required
* Data stored locally in TypeScript files
